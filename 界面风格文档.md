# 界面风格文档

## 1. 设计风格概述

本项目采用现代化的战术军事风格界面，融合了暗色主题与战术元素，营造出专业、高端的军事指挥系统氛围。整体设计强调功能性与视觉层次，同时保持简洁与专业性。

**核心设计理念：**
- 暗色主题：减少视觉疲劳，适合长时间操作
- 战术元素：网格、扫描线、军事风格图标
- 层次感：通过卡片、边框和阴影创建深度
- 响应式：适配不同屏幕尺寸
- 交互反馈：丰富的动画和过渡效果

## 4. 组件样式

### 4.1 按钮

**主要按钮（Primary）：**
- 背景色：#8b956d
- 文字色：#1a1a1a
- 圆角：0.375rem
- 字体：大写，加粗， tracking-wide
- 悬停效果：轻微上浮，背景色变亮

**次要按钮（Secondary）：**
- 背景色：透明
- 边框：1px solid #8b956d
- 文字色：#8b956d
- 悬停效果：背景色变为 #8b956d/10

**危险按钮（Danger）：**
- 背景色：#8b4444
- 文字色：#e0e0e0
- 悬停效果：轻微上浮，背景色变亮

### 4.2 卡片

- 背景色：#2a2a2a
- 边框：1px solid #3a3a3a
- 圆角：0.375rem
- 阴影：轻微阴影，悬停时增强
- 悬停效果：边框颜色变为 #8b956d，阴影增强

### 4.3 输入框

- 背景色：#1a1a1a
- 边框：1px solid #3a3a3a
- 文字色：#e0e0e0
-  placeholder 颜色：#666666
- 聚焦效果：边框变为 #8b956d，添加轻微阴影

### 4.4 威胁级别徽章

**高威胁：**
- 背景色：bg-red-500/20
- 文字色：text-red-300
- 边框：border-red-500/50

**中威胁：**
- 背景色：bg-amber-500/20
- 文字色：text-amber-300
- 边框：border-amber-500/50

**低威胁：**
- 背景色：bg-green-500/20
- 文字色：text-green-300
- 边框：border-green-500/50

## 5. 动画效果

### 5.1 常用动画

| 动画名称 | 描述 | 用途 |
|---------|------|------|
| pulse-ring | 脉冲环效果，向外扩散并淡出 | 用于目标标记的呼吸效果 |
| slideInRight | 从右侧滑入 | 用于右侧面板的进入效果 |
| slideInUp | 从下方滑入 | 用于选项卡片的进入效果 |
| fadeIn | 淡入效果 | 用于标签页内容的切换 |
| expandIn | 扩展进入效果 | 用于组件的初始加载 |
| typing | 打字效果 | 用于AI思考状态的指示 |
| scan | 扫描线效果 | 用于战术界面的扫描动画 |

### 5.2 过渡效果

- 所有交互元素都有 0.2-0.3 秒的过渡效果
- 悬停时的变换和颜色变化
- 点击时的反馈动画
- 平滑的滚动效果

## 6. 布局结构

### 6.1 整体布局

- 顶部导航栏：固定高度，包含系统标题和控制按钮
- 主内容区：分为左侧地图区域和右侧信息面板
- 左侧地图：占据主要空间，显示战术地图和目标标记
- 右侧面板：固定宽度，包含AI聊天、目标情报和打击选项
- 底部情报 feed：显示实时情报更新
- 状态栏：底部固定，显示系统状态信息

### 6.2 响应式设计

- 桌面端：完整布局，左右分栏
- 平板端：右侧面板宽度减小
- 移动端：可折叠右侧面板，优先显示地图

## 7. 交互模式

### 7.1 地图交互

- 点击目标标记：显示目标详情，更新右侧面板
- 悬停目标标记：显示放大效果和高亮
- 选择目标：地图自动定位到目标位置
- 绘制打击路线：显示从基地到目标的路线

### 7.2 聊天交互

- 输入消息：按Enter发送
- 快速操作：点击按钮执行常用命令
- AI思考状态：显示打字动画和脉冲效果
- 消息滚动：自动滚动到最新消息

### 7.3 面板交互

- 标签页切换：点击标签切换内容，带有淡入动画
- 卡片选择：点击卡片显示选中状态，带有边框和阴影变化
- 按钮点击：带有轻微上浮和颜色变化效果
- 滚动：自定义滚动条，与整体风格一致

## 8. 特殊效果

### 8.1 战术网格

- 背景网格：50px × 50px的轻微网格线
- 颜色：rgba(139, 149, 109, 0.03)
- 用途：增强战术氛围，提供空间参考

### 8.2 扫描线效果

- 水平扫描线：从顶部到底部移动
- 颜色：rgba(139, 149, 109, 0.5)
- 动画：3秒线性循环
- 用途：增强战术界面的科技感

### 8.3 坐标显示

- 位置：地图右下角
- 格式：度数，使用等宽字体
- 内容：显示当前选中目标或默认中心的坐标

### 8.4 图例

- 位置：地图左下角
- 内容：显示不同威胁级别的颜色说明
- 样式：半透明背景，边框，圆角

## 10. 设计规范

### 10.1 间距规范

- 大间距：1.5rem (24px)
- 中等间距：1rem (16px)
- 小间距：0.5rem (8px)
- 微间距：0.25rem (4px)

### 10.2 圆角规范

- 大圆角：0.75rem (12px)
- 中等圆角：0.375rem (6px)
- 小圆角：0.25rem (4px)

### 10.3 阴影规范

- 轻微阴影：0 1px 2px 0 rgb(0 0 0 / 0.05)
- 中等阴影：0 4px 12px rgba(0, 0, 0, 0.4)
- 强阴影：0 8px 24px rgba(0, 0, 0, 0.6)

### 10.4 图标规范

- 使用 Lucide React 图标库
- 图标大小：16px, 20px, 24px, 32px
- 图标颜色：根据上下文使用相应的战术颜色

## 11. 总结

本项目的界面风格采用了现代化的战术军事设计，通过暗色主题、战术元素和精心设计的交互效果，营造出专业、高端的军事指挥系统氛围。设计注重功能性与视觉层次，同时保持简洁与专业性，为用户提供直观、高效的操作体验。

通过严格遵循设计规范和使用统一的颜色、排版、组件样式，可以确保整个应用的视觉一致性和专业性，同时为后续的功能扩展和界面更新提供清晰的设计指导。
